# How to use Markdown

## Highlight
`**Bold**` **Bold**</br>
`*Emphasize*` *Emphasize*</br>
`++Underline++` ++Underline++</br>
`==Highlight==` ==Highlight==</br>
`~~Strikethorough~~` ~~Strikethorough~~</br>
`^superscript^` ^superscript^</br>
`~subscript~` ~subscript~</br>
